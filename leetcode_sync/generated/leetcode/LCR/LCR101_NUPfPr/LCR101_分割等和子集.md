## 题目描述

给定一个非空的正整数数组 `nums` ，请判断能否将这些数字分成元素和相等的两部分。

**示例 1：**

**输入：** nums = [1,5,11,5]

**输出：** true

**解释：** nums**** 可以分割成 [1, 5, 5] 和 [11] 。

**示例 2：**

**输入：** nums = [1,2,3,5]

**输出：** false

**解释：** nums**** 不可以分为和相等的两部分

**提示：**
  * `1 <= nums.length <= 200`
  * `1 <= nums[i] <= 100`
注意：本题与主站 416 题相同： <https://leetcode.cn/problems/partition-equal-subset-sum/>

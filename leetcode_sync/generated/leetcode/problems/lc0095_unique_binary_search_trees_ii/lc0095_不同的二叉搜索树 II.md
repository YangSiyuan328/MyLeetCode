## 题目描述

给你一个整数 `n` ，请你生成并返回所有由 `n` 个节点组成且节点值从 `1` 到 `n` 互不相同的不同 **二叉搜索树** __ 。可以按 **任意顺序** 返回答案。

**示例 1：**

![](https://assets.leetcode.com/uploads/2021/01/18/uniquebstn3.jpg)

**输入：** n = 3

**输出：**[[1,null,2,null,3],[1,null,3,2],[2,1,3],[3,1,null,null,2],[3,2,null,1]]

**示例 2：**

**输入：** n = 1

**输出：**[[1]]

**提示：**
  * `1 <= n <= 8`

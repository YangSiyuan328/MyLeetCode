## 题目描述

请你找到并返回这个整数

**示例：**

**输入：** arr = [1,2,2,6,6,6,6,7,10]

**输出：** 6

**提示：**
  * `1 <= arr.length <= 10^4`
  * `0 <= arr[i] <= 10^5`

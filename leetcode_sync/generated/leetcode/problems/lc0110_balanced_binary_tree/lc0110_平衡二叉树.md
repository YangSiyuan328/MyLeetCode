## 题目描述

给定一个二叉树，判断它是否是 平衡二叉树  

**示例 1：**

![](https://assets.leetcode.com/uploads/2020/10/06/balance_1.jpg)

**输入：** root = [3,9,20,null,null,15,7]

**输出：** true

**示例 2：**

![](https://assets.leetcode.com/uploads/2020/10/06/balance_2.jpg)

**输入：** root = [1,2,2,3,3,null,null,4,4]

**输出：** false

**示例 3：**

**输入：** root = []

**输出：** true

**提示：**
  * 树中的节点数在范围 `[0, 5000]` 内
  * `-104 <= Node.val <= 104`

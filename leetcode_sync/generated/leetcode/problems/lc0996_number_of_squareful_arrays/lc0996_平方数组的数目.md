## 题目描述

如果一个数组的任意两个相邻元素之和都是 **完全平方数** ，则该数组称为 **平方数组** 。
给定一个整数数组 `nums`，返回所有属于 **平方数组** 的 `nums` 的排列数量。
如果存在某个索引 `i` 使得 `perm1[i] != perm2[i]`，则认为两个排列 `perm1` 和 `perm2` 不同。

**示例 1：**

**输入：** nums = [1,17,8]

**输出：** 2

**解释：**[1,8,17] 和 [17,8,1] 是有效的排列。

**示例 2：**

**输入：** nums = [2,2,2]

**输出：** 1

**提示：**
  * `1 <= nums.length <= 12`
  * `0 <= nums[i] <= 109`

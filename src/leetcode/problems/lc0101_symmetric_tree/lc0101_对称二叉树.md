![](https://pic.leetcode.cn/1698026966-JDYPDU-image.png)

**输入：** root = [1,2,2,3,4,4,3]

**输出：** true

![](https://pic.leetcode.cn/1698027008-nPFLbM-image.png)

**输入：** root = [1,2,2,null,3,null,3]

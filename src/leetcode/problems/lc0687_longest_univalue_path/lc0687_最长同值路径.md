## 题目描述

给定一个二叉树的 `root` ，返回 _最长的路径的长度_ ，这个路径中的 _每个节点具有相同值_  。 这条路径可以经过也可以不经过根节点。

**两个节点之间的路径长度**  由它们之间的边数表示。

**示例 1:**

![](https://assets.leetcode.com/uploads/2020/10/13/ex1.jpg)

**输入：** root = [5,4,5,1,1,5]

**输出：** 2

**示例 2:**

![](https://assets.leetcode.com/uploads/2020/10/13/ex2.jpg)

**输入：** root = [1,4,5,4,4,5]

**输出：** 2

**提示:**
  * 树的节点数的范围是 `[0, 104]` 
  * `-1000 <= Node.val <= 1000`
  * 树的深度将不超过 `1000`

## 示例

```
[5,4,5,1,1,null,5]
[1,4,5,4,4,null,5]
```

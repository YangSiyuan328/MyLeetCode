## 题目描述

给定一棵二叉树的根节点 `root` ，请找出该二叉树中每一层的最大值。

**示例1：**

![](https://assets.leetcode.com/uploads/2020/08/21/largest_e1.jpg)

**输入:** root = [1,3,2,5,3,null,9]

**输出:**[1,3,9]

**示例2：**

**输入:** root = [1,2,3]

**输出:**[1,3]

**提示：**
  * 二叉树的节点个数的范围是 `[0,104]`
  * `-231 <= Node.val <= 231 - 1`

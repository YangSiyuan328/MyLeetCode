## 题目描述

给你一个整数数组 `nums` ，返回 __`nums[i] XOR nums[j]` 的最大运算结果，其中 `0 ≤ i ≤ j < n` 。

**示例 1：**

**输入：** nums = [3,10,5,25,2,8]

**输出：** 28

**解释：** 最大运算结果是 5 XOR 25 = 28.

**示例 2：**

**输入：** nums = [14,70,53,83,49,91,36,80,92,51,66,70]

**输出：** 127

**提示：**
  * `1 <= nums.length <= 2 * 105`
  * `0 <= nums[i] <= 231 - 1`
